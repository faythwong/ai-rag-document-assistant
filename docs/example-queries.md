# Example Queries

Below are example questions users can ask the AI assistant once documents have been processed.

---

## Document Summarization

```
Summarize the uploaded report.
```

---

## Information Extraction

```
What revenue figures are mentioned in the document?
```

---

## Contract Analysis

```
What does the contract say about termination clauses?
```

---

## Key Insights

```
What are the main risks mentioned in the document?
```

---

## Data Extraction

```
List the important financial metrics mentioned in the report.
```

---

These queries demonstrate how the system retrieves relevant document chunks using vector similarity search before generating responses.
